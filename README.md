# YGO/compe-frontend

## Setup

Prepare backend.
```bash
git clone https://github.com/YGO/compe-backend.git
cd compe-backend
# Follow README in compe-backend.
```

Install dependencies.
```bash
cd compe-frontend
yarn install
```

## Develop

```bash
yarn start
open http://localhost:3000/pgateaching_201709
```

Testing
```bash
yarn test:watch
```

## Deploy

Build. Built files will be under dist directory.
```bash
yarn build
```
